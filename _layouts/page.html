Cursor movement

  • Ctrl + h - move cursor left
  • Ctrl + j - move cursor down
  • Ctrl + k - move cursor up
  • Ctrl + l - move cursor right
Tip you should check out the intellij docs for more keyboard shortcuts. Remember that all commands and most controls in intellij can be customized in your settings.

Terminal

  • Ctrl + x - opens/closes a terminal window

Navigation and Tabs

  • CMD + p - fuzzy find file in project
  • CMD + e - fuzzy find recently opened file (use this over tabs)
  • CMD + t - navigate current file by file structure
  • Ctrl + v - split current window vertically
  • Ctrl + w + Ctrl + w - switch between window splits
  • CMD + up arrow - open file navigation (you can fuzzy search, navigate with arrows, and do all of the shortcuts that you can in the file tree)
Tip tabs are turned off visually.
